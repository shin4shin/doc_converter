============================================
  문서 변환기 - 사용 안내
============================================

■ 사전 준비 (최초 1회만)

  LibreOffice 설치가 필요합니다.
  아래 주소에서 다운로드 후 설치해주세요.

  → https://www.libreoffice.org/download/download/

  ※ 설치 시 "한국어" 선택 권장


■ 실행 방법

  문서변환기.exe 를 더블클릭하면 바로 실행됩니다.
  Python 설치 불필요.


■ 지원 변환

  HWP  → PDF, DOCX, TXT
  HWPX → PDF, DOCX, TXT
  DOCX → PDF, TXT, HWP, HWPX
  PDF  → TXT, DOCX, HWP, HWPX


■ 사용 방법

  1. [열기] 버튼으로 변환할 파일 선택
  2. 변환할 형식 버튼 클릭 (PDF / DOCX 등)
  3. 저장 경로 지정 (비우면 원본과 같은 폴더에 저장)
  4. [변환 시작 ▶] 클릭
  5. 완료 후 저장 폴더가 자동으로 열립니다


■ 문의

  변환 실패 시 LibreOffice 가 정상 설치되었는지 확인해주세요.
